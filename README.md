teamforge Cookbook
==================

Installs and configures CollabNet TeamForge Server or the command line client.

Requirements
------------

TeamForge Server is supported on RedHat Enterprise Linux 6.x, CentOS 6.x, or SuSE Linux Enterprise Server 11.
